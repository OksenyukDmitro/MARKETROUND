## Available Scripts

### Start the app

`npm run start`

### Lint code

`npm run lint`

`npm run lint:fix` - lint with autofix

### Test

`npm run test`

### Release

Scripts to do version bump (semver) and CHANGELOG generation 

- `npm run release:major` - bumps major

- `npm run release:minor` - bumps minor

- `npm run release:patch` - bumps patch
